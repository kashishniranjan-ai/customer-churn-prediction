# Customer Churn Prediction System

A production-ready, end-to-end Machine Learning web application for predicting Telco customer churn in real time, built with Flask, XGBoost, and a fully responsive modern dashboard UI.

---

## Features

- **XGBoost model** trained on the Telco Customer Churn dataset (30 features, one-hot encoded via `pandas.get_dummies(drop_first=True)`)
- **Real-time predictions** via a Flask REST API (`/predict`)
- **Animated gauge + risk bar** for visual probability communication
- **Feature importance chart** (Chart.js horizontal bar, top-15 features)
- **Confusion matrix display** with model test accuracy
- **Dark mode toggle** with `localStorage` persistence
- **Form validation** with clear error messages
- **Loading spinner** during inference
- **Fully responsive** — desktop, tablet, and mobile

## Quick Start

### 1. Install dependencies

```bash
pip install -r requirements.txt
```

### 2. Train the model

```bash
python train_model.py
```

This generates:
- `churn_model.pkl`
- `static/feature_importances.json`
- `static/confusion_matrix.json`

### 3. Run the app

```bash
python app.py
```

Open your browser at **http://localhost:5000**

## Using the Real Telco Dataset

Download the [IBM Telco Customer Churn dataset](https://www.kaggle.com/blastchar/telco-customer-churn) from Kaggle, then replace the data generation section in `train_model.py`:

```python
# Replace synthetic data generation with:
data = pd.read_csv("WA_Fn-UseC_-Telco-Customer-Churn.csv")
```

## API Reference

### `POST /predict`

Accepts `multipart/form-data` with all 19 customer feature fields.

**Response**

```json
{
  "prediction":  1,
  "will_churn":  true,
  "probability": 72.45,
  "label":       "Customer Will Churn",
  "status":      "danger"
}
```

### `GET /feature-importances`

Returns a JSON object of the top-15 XGBoost feature importances (sorted descending).

### `GET /confusion-matrix`

Returns the confusion matrix values (tn, fp, fn, tp) and overall test accuracy from training.
